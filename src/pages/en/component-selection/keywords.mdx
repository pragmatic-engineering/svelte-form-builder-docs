Components can searched by their name or Keywords

<img src="/component-selection/keywords.png" />

You can update components Keywords or attach your own to custom components

```ts
let options: BuilderOptions = {
  componentOptions: [
    {
      componentName: "Text",
      keywords: ["Hello"],
    },
    {
      componentName: "Audio",
      keywords: ["Music"],
    },
  ],
};
```
