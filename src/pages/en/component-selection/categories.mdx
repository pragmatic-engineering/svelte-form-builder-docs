Components can be grouped into Categories

<img src="/component-selection/categories.png" />

You can override which components belong to which Category or create your own categories

```ts
let options: BuilderOptions = {
  componentOptions: [
    {
      componentName: "Text",
      categories: ["Custom Category"],
    },
  ],
};
```

- Open categories, you should see the Custom Category
