### Generates

A standard HTML button

### Unique Properties

- <a href="https://developer.mozilla.org/en-US/docs/Web/HTML/Element/button#attr-type" target="_blank">
    Type
  </a>
  - Default is button.

### Tips & Notes

- Use the standard events property to handle a click event

```ts
let options: BuilderOptions = {
  formDefinition: [],
  componentOptions: [
    {
      componentName: "Button",
      events: {
        onclick: (e: Event) => {
          alert(`Hello from ${e.target.id}`);
        },
      },
    },
  ],
};
```
