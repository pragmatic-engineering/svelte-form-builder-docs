### Property

### Purpose

- Provide a way to create custom Build tool buttons

```ts
{
  customBuildTools: [
    {
      buttonText: 'Surprise Me',
      onClick: (params) => {
        return `<marquee>✨ Alive and kicking! ✨</marquee>`;
      }
    }
  ],
}
```
