These Tools can be used while the **BUILD** view is active

<img src="/tools/build-tools.png" />

- Show Definition
  Shows the entire definition

- Validate Definition
  Test any custom definition validation

- Export Definition
  Returns a json file of the form definition

- Import Definition
  Imports a form definition from a previous file export

- Clear Current Tab
  Clears the current active tab of all fields

- Clear All Tabs
  Clears all tabs
